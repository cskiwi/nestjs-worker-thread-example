## Description

This project illustrates how to use node.js worker thread in a NestJS app and have access to dependency injection.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
